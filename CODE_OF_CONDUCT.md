# Contributor Covenant Code of Conduct

We pledge to make participation in this project a harassment-free experience for everyone. Contributors must use welcoming and inclusive language, respect differing viewpoints, accept constructive criticism, focus on what is best for learners and communities, and show empathy.

Unacceptable behavior includes harassment, discriminatory profiling, publication of private information, and using this project to justify automated high-stakes decisions about students. Project maintainers may remove or reject contributions and participation that violate these expectations.

Report conduct concerns privately to the repository owner through the GitHub profile. Maintainers will investigate proportionately and protect the privacy of reporters.
